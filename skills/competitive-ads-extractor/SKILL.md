# 競爭廣告提取器

此技能從廣告庫中提取競爭對手的廣告，並分析哪些有效——他們突出的問題、針對的使用案例，以及引起共鳴的文案/創意。

## 何時使用此技能

- 研究競爭對手的廣告策略
- 為您自己的廣告尋找靈感
- 了解市場定位
- 識別成功的廣告模式
- 分析有效的訊息傳遞
- 發現新的使用案例或痛點
- 使用經過驗證的概念規劃廣告活動

## 此技能的功能

1. **提取廣告**：從 Facebook Ad Library、LinkedIn 等抓取廣告
2. **擷取截圖**：保存所有廣告的視覺副本
3. **分析訊息**：識別問題、使用案例和價值主張
4. **分類廣告**：按主題、受眾或格式分組
5. **識別模式**：找出常見的成功方法
6. **提供洞察**：解釋為什麼某些廣告可能表現良好

## 範例

**使用者**："從 Facebook Ad Library 提取 Notion 的廣告，並告訴我哪些訊息對他們有效。"

**處理過程**：
```
正在訪問 Facebook Ad Library...
搜索：Notion
找到：23 個活躍廣告

正在提取截圖...
[████████████████████] 100%

保存至：competitor-ads/notion/
- ad-001-collaboration.png
- ad-002-productivity.png
- ad-003-templates.png
...

正在分析訊息...
```

**輸出**：
```
# Notion 廣告分析

## 概覽
- 總廣告數：23 個活躍
- 主要主題：生產力 (35%)、協作 (30%)、
  模板 (20%)、AI 功能 (15%)
- 廣告格式：靜態圖片 (60%)、影片 (40%)
- CTA 模式："免費試用"、"開始使用"

## 他們突出的關鍵問題

1. **資訊分散** (8 個廣告)
   文案："停止在 10 個不同工具之間切換"
   為何有效：直接針對許多團隊面臨的痛點

2. **會議過載** (5 個廣告)
   文案："用異步更新取代不必要的會議"
   為何有效：疫情後遠程工作的痛點

3. **文件遺失** (4 個廣告)
   文案："再也不用問'那份文件在哪裡？'"
   為何有效：普遍的職場挫折

## 成功的創意模式

### 模式 1：前後對比
- 展示混亂的工具環境 → 整潔的 Notion 工作空間
- 用於 6 個高績效廣告
- 視覺隱喻立即清晰

### 模式 2：功能展示
- 實際產品使用的 GIF
- 在 5 秒內展示特定功能
- 用於新功能（AI、模板）

### 模式 3：社會證明
- "加入 2000 萬用戶"的訊息
- 客戶標誌
- 用於 4 個針對企業的廣告

## 有效的文案

最佳標題：
1. "您團隊的知識，終於集中在一個地方"
   → 以利益為中心，直接針對痛點

2. "多合一工作空間"
   → 清晰定位，廣泛吸引力

3. "真正幫助您工作的 AI"
   → 針對 AI 懷疑論，實用角度

最佳正文文案模式：
- 短句（少於 10 個字）
- 專注於結果而非功能
- 包含具體數字（"減少 50% 的會議"）

## 受眾定位洞察

基於廣告變化：
- 創業創始人：個人生產力角度
- 團隊負責人：協作和對齊
- 企業：提及安全性和合規性
- 學生：免費計劃、模板、組織

## 對您廣告的建議

1. **測試"工具泛濫"痛點**
   → 基於他們的廣告頻率，有強烈共鳴

2. **使用產品截圖而非抽象視覺**
   → 他們所有的頂級廣告都展示實際 UI

3. **以問題而非解決方案引導**
   → "厭倦了 X？"比"推出 Y"表現更好

4. **保持文案在 100 個字符以下**
   → 他們最短的廣告似乎最常見

5. **測試前後視覺格式**
   → 在他們的創意中驗證的模式

## 已保存的檔案
- 所有廣告：~/competitor-ads/notion/
- 分析：~/competitor-ads/notion/analysis.md
- 最佳表現者：~/competitor-ads/notion/top-10/
```

## 您可以學到什麼

### 訊息分析

- 目標受眾細分

### 創意模式
- 有效的視覺風格
- 影片與靜態圖片的表現
- 配色方案和品牌
- 佈局模式

### 文案公式
- 標題結構
- 行動呼籲模式
- 長度和語氣
- 情緒觸發

### 活動策略
- 季節性活動
- 產品發布方法
- 功能公告策略
- 重新定位模式

## 最佳實踐

### 法律與道德
✓ 僅用於研究和靈感
✓ 不要直接複製廣告
✓ 尊重知識產權
✓ 使用洞察來告知原創創意
✗ 不要抄襲文案或竊取設計

### 分析技巧
1. **尋找模式**：哪些主題重複出現？
2. **隨時間追蹤**：每月保存廣告以查看演變
3. **測試假設**：為您的品牌調整成功模式
4. **按受眾細分**：不同目標的不同訊息
5. **比較平台**：LinkedIn 與 Facebook 的訊息不同

## 常見工作流程

### 廣告活動規劃
1. 提取競爭對手廣告
2. 識別成功模式
3. 注意他們訊息中的空白
4. 頭腦風暴獨特角度
5. 草擬測試廣告變化

### 定位研究
1. 獲取 5 個競爭對手的廣告
2. 繪製他們的定位
3. 找出服務不足的角度
4. 開發差異化訊息
5. 針對他們的方法進行測試

### 創意靈感
1. 按主題提取廣告
2. 分析視覺模式
3. 注意顏色和佈局趨勢
4. 調整成功模式
5. 創建原創變化

## 成功技巧

1. **定期監控**：每月檢查變化
2. **廣泛研究**：也關注相鄰競爭對手
3. **保存所有內容**：建立參考庫
4. **測試洞察**：運行您自己的實驗
5. **追蹤表現**：A/B 測試受啟發的概念
6. **保持原創**：用於靈感，而非複製
7. **多平台**：比較 Facebook、LinkedIn、TikTok 等

## 輸出格式

- **截圖**：所有廣告保存為圖片
- **分析報告**：洞察的 Markdown 摘要
- **試算表**：包含廣告文案、CTA、主題的 CSV
- **簡報**：頂級表現者的視覺簡報
- **模式庫**：按方法分類
