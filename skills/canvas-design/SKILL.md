這些是創建設計理念的說明 - 美學運動隨後會以視覺方式表達。僅輸出 .md 檔案、.pdf 檔案和 .png 檔案。

分兩個步驟完成：
1. 設計理念創作（.md 檔案）
2. 在畫布上表達創作（.pdf 檔案或 .png 檔案）

首先，執行此任務：

## 設計理念創作

首先，創建一個視覺理念（不是版面配置或範本），它將通過以下方式詮釋：
- 形式、空間、顏色、構圖
- 圖像、圖形、形狀、圖案
- 最少的文字作為視覺點綴

### 關鍵理解
- 收到的內容：使用者提供的一些微妙的輸入或指示，應該加以考慮，但作為基礎；它不應限制創作自由。
- 創作的內容：設計理念/美學運動。
- 接下來發生什麼：然後，同一版本接收理念並以視覺方式表達 - 創作 90% 視覺設計、10% 必要文字的成品。

考慮這種方法：
- 為藝術運動撰寫宣言
- 下一階段涉及創作藝術品

理念必須強調：視覺表達。空間溝通。藝術詮釋。最少文字。

### 如何生成視覺理念

**命名運動**（1-2 個詞）：「粗獷主義的喜悅」/「色彩的沉默」/「新陳代謝派的夢想」

**闡述理念**（4-6 段 - 簡潔但完整）：

要捕捉視覺精髓，表達理念如何通過以下方式呈現：
- 空間與形式
- 顏色與材質
- 比例與節奏
- 構圖與平衡
- 視覺層次

**關鍵指南：**
- **避免冗餘**：每個設計方面應該只提及一次。避免重複關於色彩理論、空間關係或排版原則的觀點，除非增加新的深度。
- **反覆強調工藝**：理念必須多次強調最終作品應該看起來像是花費了無數小時創作的，經過精心打磨，來自該領域絕對頂尖的人。這種框架是必不可少的 - 重複使用「精心打造」、「深厚專業知識的產物」、「細緻入微的關注」、「大師級執行」等詞語。
- **留下創作空間**：對美學方向保持具體，但足夠簡潔，以便下一個 Claude 有空間做出同樣極高工藝水準的詮釋選擇。

理念必須引導下一個版本以視覺方式表達想法，而不是通過文字。資訊存在於設計中，而不是段落中。

### 理念範例

**「具體詩」**
理念：通過紀念性形式和大膽幾何進行溝通。
視覺表達：巨大的色塊、雕塑般的排版（巨大的單字、微小的標籤）、粗獷主義的空間劃分、波蘭海報的能量遇上柯比意。想法通過視覺重量和空間張力表達，而非解釋。文字作為罕見、有力的姿態 - 從不是段落，只有融入視覺建築的必要詞語。每個元素都以大師工匠的精確度放置。

**「色彩語言」**
理念：顏色作為主要資訊系統。
視覺表達：幾何精確，色彩區域創造意義。排版最小化 - 小型無襯線標籤讓色彩領域進行溝通。想像約瑟夫・阿爾伯斯的互動遇上資料視覺化。資訊以空間和色彩編碼。文字僅用於錨定顏色已經展示的內容。細緻色彩校準的結果。

**「類比冥想」**
理念：通過紋理和呼吸空間進行安靜的視覺沉思。
視覺表達：紙張紋理、墨水暈染、廣闊的負空間。攝影和插圖佔主導。排版低語（小、克制、服務於視覺）。日本攝影書美學。圖像在頁面上呼吸。文字稀少出現 - 短語，從不解釋性的段落。每個構圖都以冥想練習的關注來平衡。

**「有機系統」**
理念：自然聚集和模組化生長模式。
視覺表達：圓潤的形式、有機的排列、通過建築從自然中獲取的顏色。資訊通過視覺圖表、空間關係、圖標顯示。文字僅用於漂浮在空間中的關鍵標籤。構圖通過專業的空間編排講述故事。

**「幾何寂靜」**
理念：純粹的秩序和克制。
視覺表達：基於網格的精確、大膽的攝影或鮮明的圖形、戲劇性的負空間。排版精確但最小化 - 小型必要文字、大型安靜區域。瑞士形式主義遇上粗獷主義的材質誠實。結構在溝通，而非文字。每一個對齊都是無數次改進的成果。

*這些是濃縮的範例。實際的設計理念應該是 4-6 段實質性的內容。*

### 基本原則
- **視覺理念**：創建一個通過設計表達的美學世界觀
- **最少文字**：始終強調文字稀少、僅限必要、作為視覺元素整合 - 從不冗長
- **空間表達**：想法通過空間、形式、顏色、構圖來溝通 - 而非段落
- **藝術自由**：下一個 Claude 以視覺方式詮釋理念 - 提供創作空間
- **純粹設計**：這是關於製作藝術品，而非裝飾過的文件
- **專業工藝**：反覆強調最終作品必須看起來經過精心打造、細心雕琢，是該領域頂尖人士無數小時工作的產物

**設計理念應該有 4-6 段。** 用詩意的設計理念填滿它，將核心願景整合在一起。避免重複相同的觀點。保持設計理念的通用性，不提及藝術的意圖，就好像它可以用於任何地方。將設計理念輸出為 .md 檔案。

## 推導微妙的參考

**關鍵步驟**：在創建畫布之前，從原始請求中識別微妙的概念線索。

**基本原則**：
主題是**嵌入在藝術本身中的微妙、小眾參考** - 並非總是字面的，始終是精緻的。熟悉該主題的人應該能直覺地感受到它，而其他人只是體驗到一個精湛的抽象構圖。設計理念提供美學語言。推導出的主題提供靈魂 - 無形地編織到形式、顏色和構圖中的安靜概念 DNA。

這**非常重要**：參考必須經過提煉，使其增強作品的深度而不自我宣告。像爵士音樂家引用另一首歌一樣思考 - 只有知道的人才會注意到，但每個人都欣賞音樂。

## 畫布創作

在確立了理念和概念框架後，在畫布上表達它。花一點時間整理思緒並清空頭腦。使用創建的設計理念和以下說明來打造傑作，以專業工藝體現理念的所有方面。

**重要**：對於任何類型的內容，即使使用者要求為電影/遊戲/書籍創作，方法仍應該是精緻的。永遠不要忘記這應該是藝術，而不是卡通或業餘的東西。

要創作博物館或雜誌品質的作品，以設計理念為基礎。創建一個高度視覺化、設計導向的單頁 PDF 或 PNG 輸出（除非被要求更多頁）。通常使用重複的圖案和完美的形狀。將抽象的哲學設計視為科學聖經，借用系統觀察的視覺語言 - 密集累積的標記、重複的元素或通過耐心重複建立意義並獎勵持續觀看的分層圖案。添加稀疏、臨床的排版和系統參考標記，暗示這可能是來自虛構學科的圖表，以通常保留給記錄可觀察現象的同樣敬意對待不可見的主題。用簡單的短語或細節作為錨點，巧妙地定位，使用有意圖且連貫的有限色彩調色板。擁抱使用分析性視覺語言表達人類經驗想法的悖論：結果應該感覺像是一件證明短暫事物可以被研究、繪製和通過細心關注理解的成品。這是真正的藝術。

**文字作為情境元素**：文字始終是最小化和視覺優先的，但讓情境引導這意味著低語般安靜的標籤還是大膽的排版姿態。龐克場地海報可能比極簡陶藝工作室標識有更大、更具侵略性的字體。大多數時候，字體應該是細的。所有字體的使用必須是設計導向的，優先考慮視覺溝通。無論文字大小如何，沒有任何東西掉出頁面，沒有任何東西重疊。每個元素必須包含在畫布邊界內，具有適當的邊距。仔細檢查所有文字、圖形和視覺元素都有呼吸空間和清晰的分離。這對於專業執行是不可協商的。**重要：如果寫文字，請使用不同的字體。搜尋 `~/.claude/skills/canvas-design/canvas-fonts` 目錄。無論採用何種方法，精緻是不可協商的。**

下載並使用任何需要的字體來實現這一點。通過使排版真正成為藝術本身的一部分來發揮創意 -- 如果藝術是抽象的，將字體帶到畫布上，而不是數位排版。

要突破界限，遵循設計直覺/本能，同時以理念為指導原則。擁抱終極設計自由和選擇。將美學和設計推向前沿。

**關鍵**：要達到人工打造的品質（而非 AI 生成），創作看起來花費了無數小時的作品。讓它看起來像是該領域絕對頂尖的人細心雕琢每一個細節。確保構圖、間距、顏色選擇、排版 - 一切都散發出專家級的工藝。仔細檢查沒有任何東西重疊，格式完美，每個細節都完美。創作可以展示給人們以證明專業知識並被評為無可否認的令人印象深刻的作品。

將最終結果輸出為單個可下載的 .pdf 或 .png 檔案，連同使用的設計理念作為 .md 檔案。

## 最後步驟

**重要**：使用者已經說過「還不夠完美。它必須是原始的，是工藝的傑作，就像即將在博物館展出一樣。」

**關鍵**：要完善作品，避免添加更多圖形；而是完善已創建的內容，使其極其清晰，完全尊重設計理念和極簡主義原則。與其添加有趣的濾鏡或重構字體，不如考慮如何使現有構圖更與藝術融為一體。如果本能是調用新函數或繪製新形狀，請停下來，而是問：「如何使現有的東西更像藝術品？」

進行第二輪。回到程式碼中進一步完善/打磨，使其成為一個哲學設計的傑作。

## 多頁選項

當被要求創建額外頁面時，沿著設計理念的相同路線創建更多創意頁面，但也要明顯不同。將這些頁面打包在同一個 .pdf 或多個 .png 中。將第一頁視為等待填滿的整本咖啡桌書籍中的一頁。使後續頁面成為原作的獨特變奏和回憶。讓它們以非常有品味的方式幾乎講述一個故事。發揮完全的創作自由。
