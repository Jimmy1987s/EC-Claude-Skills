# 黏土渲染 (Clay Render)

## 基本資訊

| 項目 | 說明 |
|------|------|
| 名稱 | 黏土渲染 |
| 英文 | Clay Render |
| 調性 | 可愛、箱庭風格、趣味性、整齊有序 |

## 視覺識別

| 項目 | 設定值 |
|------|--------|
| 背景色 | 柔和米色 (#F5F5DC) 或 天空藍 (#E0F7FA) |
| 文字色 | 巧克力棕 (#5D4037) 或 深灰色 (#424242) |
| 強調色 | 鮮豔橙色 (#FF9800) 或 萊姆綠 (#CDDC39) |

## 圖像風格

- **特徵：** 方塊感的低多邊形 3D 模型、等角投影俯瞰圖
- **形狀：** 多邊形、方塊
- **質感：** 霧面黏土質地、略帶粗糙的表面

## 字體排版

| 用途 | 說明 |
|------|------|
| 標題 | 圓潤的粗體字型 |   
| 內文 | 親切的無襯線體 |
| 數字 | 積木風格的方塊字體 |
