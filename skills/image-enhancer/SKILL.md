# 圖片增強器

此技能讓您的圖片和螢幕截圖看起來更好——更銳利、更清晰、更專業。

## 何時使用此技能

- 改善部落格文章或文件的螢幕截圖品質
- 在社群媒體上分享前增強圖片
- 為簡報或報告準備圖片
- 放大低解析度圖片
- 銳化模糊照片
- 清理壓縮圖片

## 此技能的功能

1. **分析圖片品質**：檢查解析度、銳利度和壓縮瑕疵
2. **增強解析度**：智慧放大圖片
3. **改善銳利度**：增強邊緣和細節
4. **減少瑕疵**：清理壓縮瑕疵和雜訊
5. **針對使用情況優化**：根據預期用途（網頁、印刷、社群媒體）進行調整

## 範例

**使用者**：「改善 screenshot-2024.png 的圖片品質」

**輸出**：
```
正在分析 screenshot-2024.png...

當前規格：
- 解析度：1920x1080
- 格式：PNG
- 品質：良好，但稍微模糊

應用的增強：
✓ 放大至 2560x1440（retina）
✓ 銳化邊緣
✓ 增強文字清晰度
✓ 優化檔案大小

儲存為：screenshot-2024-enhanced.png
原始檔案保留為：screenshot-2024-original.png
```

**靈感來源：** Lenny Rachitsky 在其電子報中的工作流程 - 用於他文章中的螢幕截圖

## 技巧

- 始終保留原始檔案作為備份
- 最適合螢幕截圖和數位圖片
- 可以批次處理整個資料夾
- 如有需要，指定輸出格式（PNG 用於品質，JPG 用於較小大小）
- 對於社群媒體，提及平台以獲得最佳尺寸

## 常見使用案例

- **部落格文章**：發布前增強螢幕截圖
- **文件**：使 UI 螢幕截圖極其清晰
- **社群媒體**：為 Twitter、LinkedIn、Instagram 優化圖片
- **簡報**：為大螢幕放大圖片
- **印刷材料**：提高實體媒體的解析度
