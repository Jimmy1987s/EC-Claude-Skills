# 主題工廠技能

此技能提供精心策劃的專業字型和顏色主題集合，每個主題都有精心挑選的調色板和字型配對。選擇主題後，可以將其應用於任何成品。

## 目的

要為簡報投影片或其他成品應用一致、專業的樣式，請使用此技能。每個主題包括：
- 帶有十六進位色碼的統一調色板
- 用於標題和內文的互補字型配對
- 適合不同情境和受眾的獨特視覺識別

## 使用說明

要為投影片或其他成品應用樣式：

1. **展示主題展覽**：顯示 `theme-showcase.pdf` 檔案，讓使用者可以直觀地看到所有可用主題。不要對其進行任何修改；只需顯示檔案供查看。
2. **詢問他們的選擇**：詢問要應用哪個主題到投影片
3. **等待選擇**：獲得關於所選主題的明確確認
4. **應用主題**：選擇主題後，將所選主題的顏色和字型應用到投影片/成品

## 主題詳情

每個主題都在 `themes/` 目錄中定義，包含完整規格，包括：
- 帶有十六進位色碼的統一調色板
- 用於標題和內文的互補字型配對
- 適合不同情境和受眾的獨特視覺識別

## 應用流程

選擇偏好的主題後：
1. 從 `themes/` 目錄讀取相應的主題檔案
2. 在整個投影片中一致地應用指定的顏色和字型
3. 確保適當的對比度和可讀性
4. 在所有投影片中保持主題的視覺識別

## 創建自訂主題

要處理現有主題都不適合成品的情況，請創建自訂主題。根據提供的輸入，生成類似於上述主題的新主題。給主題一個類似的名稱，描述字型/顏色組合所代表的內容。使用提供的任何基本描述來選擇適當的顏色/字型。生成主題後，展示以供審查和驗證。之後，如上所述應用主題。
