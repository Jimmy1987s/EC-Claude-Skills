## 工作流程

### 製作向性指標月報

```
使用者：「幫我做 1 月向性指標」

Step 1：讀取上月簡報
        └─ 自動找到最近一期向性指標簡報
        └─ 擷取文字內容作為基準

Step 2：收集本月數據
        └─ 詢問：「請提供本月的 KPI 數據」
        └─ 或讀取使用者提供的資料來源

Step 3：生成比較分析
        └─ 自動計算 MoM（月比月）變化
        └─ 自動計算 YoY（年比年）變化
        └─ 標記異常指標（↑↓ 超過 10%）

Step 4：呈現版面草稿（ASCII）
        └─ 先用文字版 UI 呈現，等待確認

Step 5：製作簡報
        └─ 使用 dafong-presentation-style 風格
        └─ 產生 .pptx 檔案

Step 6：驗證輸出
        └─ 產生縮圖確認版面
```

### 數據分析

```
使用者：「分析廢車業務近半年趨勢」

Step 1：讀取歷史簡報
        └─ 自動讀取近 6 個月的向性指標簡報

Step 2：擷取 KPI 數據
        └─ 流量、轉換率、案件數、成交率

Step 3：生成分析報告
        ┌─ 趨勢圖（ASCII 或建議圖表類型）
        ├─ 同期比較表
        ├─ 異常診斷
        └─ 策略建議
```

## 常用 KPI 定義

### 廢車業務

| 指標 | 定義 | 計算方式 | 健康範圍 |
|------|------|---------|---------|
| 流量 | 網站訪問數 | GA 數據 | 持續成長 |
| 轉換率 | 流量→案件 | 案件數/流量 | 6-8% |
| 案件數 | 總進件量 | 系統統計 | 持續成長 |
| 成交率 | 案件→成交 | 成交數/案件數 | 40-50% |
| 回收台數 | 實際回收 | 系統統計 | 年目標 24萬 |
| CPA | 每案成本 | 廣告費/案件數 | <200元 |

### 企業業務

| 指標 | 定義 | 健康範圍 |
|------|------|---------|
| 成交率 | 報價→成交 | 持續提升 |
| 淨利 | 月淨利潤 | 目標 180萬/月 |
| 大案金額 | 單案 >50萬 | 季度 2-3 案 |

## 注意事項

1. **先確認再動手**：版面設計先用 ASCII 呈現，確認後再製作
2. **自動讀取歷史**：會自動參照主管會議資料夾的歷史簡報
3. **草稿存放位置**：產出的文字檔存放在 `草稿/` 資料夾
4. **備份原檔**：修改既有簡報前先備份
