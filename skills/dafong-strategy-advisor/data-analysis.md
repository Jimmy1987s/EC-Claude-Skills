## 自動讀取歷史數據

### 資料來源定位

```python

檔名格式：4.{年份}年 向性指標 - 電子商務部{YYYYMM}.pptx
```

```
1. 確定目標月份
2. 計算前期路徑（上月、去年同期）

5. 建立比較表
```

## KPI 解析規則

### 廢車業務 KPI 擷取

從簡報文字中尋找以下模式：

```
流量：搜尋 "流量 X 萬" 或 "流量X萬"
轉換率：搜尋 "轉換率 X%" 或 "X%"（在轉換率附近）
案件數：搜尋 "X月案件 X件" 或 "案件 X件"
成交率：搜尋 "成交率 X%"
回收台數：搜尋 "回收X台" 或 "回收 X 台"
CPA：搜尋 "CPA X元" 或 "CPA為X元"
```

### 數據正規化

```
單位轉換：
- 「8萬」→ 80000
- 「19,266台」→ 19266
- 「44.77%」→ 44.77
- 「+2.65%」→ 2.65（成長）
- 「-1.5%」→ -1.5（衰退）

百分點 vs 百分比：
- 成交率從 40% 到 45% = +5pp（百分點）
- 成交率成長 = +12.5%（百分比）
```

## 比較分析方法

### MoM（Month over Month）月比月

```
計算公式：(本月 - 上月) / 上月 × 100%

範例：
- 本月成交率：44.77%
- 上月成交率：43.57%
- MoM = (44.77 - 43.57) / 43.57 × 100% = +2.76%
- 或表示為：+1.2pp（百分點）
```

### YoY（Year over Year）年比年

```
計算公式：(本期 - 去年同期) / 去年同期 × 100%

範例：
- 2025年回收：19,266台
- 2024年回收：18,769台
- YoY = (19266 - 18769) / 18769 × 100% = +2.65%
```

### 累計比較

```
YTD（Year to Date）年初至今：
- 2025年1-12月累計 vs 2024年1-12月累計

QoQ（Quarter over Quarter）季比季：
- Q4 vs Q3
```

## 異常檢測規則

### 警示閾值

| 指標 | 正常範圍 | 警示條件 |
|------|---------|---------|
| 流量 MoM | -5% ~ +10% | < -10% 或 > +20% |
| 轉換率 | 5% ~ 8% | < 5% 或 > 10% |
| 成交率 | 35% ~ 50% | < 30% 或 > 55% |
| CPA | < 200元 | > 200元 |
| 回收台數 MoM | -10% ~ +15% | < -15% |

### 異常類型

```
┌─ 異常類型 ──────────────────────────────────────────────────────┐
│                                                                 │
│  📈 正向異常（需驗證是否可持續）                                  │
│  ├─ 成交率突然大幅上升（>5pp）                                  │
│  ├─ 流量暴增（>30%）                                           │
│  └─ CPA 大幅下降（>20%）                                       │
│                                                                 │
│  📉 負向異常（需立即關注）                                       │
│  ├─ 轉換率下降（>1pp）                                         │
│  ├─ 成交率下降（>3pp）                                         │
│  ├─ CPA 上升（>15%）                                           │
│  └─ 回收台數下降（>10%）                                       │
│                                                                 │
│  🔄 結構性變化（需分析原因）                                     │
│  ├─ 流量來源結構改變                                           │
│  ├─ 客戶類型比例變化                                           │
│  └─ 全服務 vs 一般服務比例變化                                  │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 趨勢分析

### 移動平均

```
3個月移動平均：
MA3 = (M1 + M2 + M3) / 3

用途：平滑短期波動，識別長期趨勢
```

### 趨勢判斷

```
上升趨勢：連續 3 個月 MoM > 0
下降趨勢：連續 3 個月 MoM < 0
持平：MoM 在 ±2% 之間波動
```

### ASCII 趨勢圖

```
成交率 (%)

44┤                  ●     ●
42┤            ●

      7月  8月  9月  10月 11月 12月

圖例：● 實際值  --- 目標線  ═══ 移動平均
```

## 分析報告生成

### 自動摘要規則

```
1. 先說結論
   - 整體表現：優於/持平/低於 預期
   - 關鍵亮點：1-2 個最重要的成就
   - 關鍵警示：1-2 個需要關注的問題

2. 再給數據
   - 核心 KPI 表格
   - 與上期/去年的比較

3. 最後建議
   - 維持/強化的策略
   - 需要調整的策略
   - 具體行動項目
```

### 報告長度建議

| 報告類型 | 長度 | 重點 |
|---------|------|------|
| 快速摘要 | 3-5 句 | 結論 + 關鍵數字 |
| 標準報告 | 1-2 頁 | 完整分析 + 建議 |
| 深度分析 | 3-5 頁 | 多維度分析 + 選項評估 |

## 數據視覺化建議

### 適用圖表類型

| 分析目的 | 建議圖表 |
|---------|---------|
| 單一指標趨勢 | 折線圖 |
| 多指標比較 | 長條圖 |
| 佔比分析 | 圓餅圖/環形圖 |
| 目標達成 | 儀表板/進度條 |
| 多維度比較 | 雷達圖 |
| 分布分析 | 直方圖 |

### KPI 卡片設計

```
┌─────────────────┐
│    [指標名稱]    │
│                 │
│      X.XX       │  ← 主要數值（大字）
│                 │
│   vs上月 +X%    │  ← 比較（小字+顏色）
│   ↑ 或 ↓        │  ← 方向指示
└─────────────────┘

顏色規則：
- 綠色：正向變化
- 紅色：負向變化
- 灰色：持平
```

## 常見分析場景

### 場景 1：月度報告

```
輸入：「分析 12 月表現」
輸出：
1. 讀取 12 月簡報
2. 讀取 11 月簡報（MoM）
3. 讀取去年 12 月簡報（YoY）
4. 生成比較表
5. 標記異常
6. 提供建議
```

### 場景 2：季度檢討

```
輸入：「Q4 季度檢討」
輸出：
1. 讀取 10、11、12 月簡報
2. 計算季度平均/累計
3. 與 Q3 比較
4. 與去年 Q4 比較
5. 識別季度趨勢
6. 策略建議
```

### 場景 3：年度總結

```
輸入：「2025 年度總結」
輸出：
1. 讀取全年 12 份簡報
2. 計算年度累計
3. 月度趨勢圖
4. 與 2024 年比較
5. 目標達成率
6. 來年建議
```

### 場景 4：問題診斷

```
輸入：「為什麼轉換率下降？」
輸出：
1. 確認下降幅度和時間範圍
2. 分析可能原因
   - 流量來源變化
   - 競爭對手活動
   - 網站/服務問題
   - 季節性因素
3. 建議診斷步驟
4. 提出改善建議
```
