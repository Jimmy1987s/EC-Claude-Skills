# 智慧模板庫

## 向性指標月報模板

### 標準結構

```
投影片 1：封面
├── 年份月份
├── 「向性指標說明」
└── 單位名稱：電子商務部

投影片 2：廢車標題頁
└── 「廢車」

投影片 3：廢車 OKR 總覽
├── Object + Goal
├── Strategy
├── Measure (Dashboard)
└── Plans

投影片 4：廢車執行狀況
├── 衡量指標（KPI 卡片）
└── 策略執行（項目清單）

投影片 5-6：廢車圖表/截圖（如有）

投影片 7：企業標題頁
└── 「企業」

投影片 8：企業 OKR 總覽

投影片 9：企業執行狀況

投影片 10：新業務標題頁
└── 「車材 & 醫廢 & 紡織 & 新業務」

投影片 11：新業務 OKR 總覽
```

### 執行狀況頁模板

**廢車業務**

```markdown
## 衡量指標
- 流量 [X] 萬
- 轉換率 [X]% ([上月X]%；[前月X]%)
- [月份]月案件 [X]件／成交 [X]件(成交率 [X]%) (全服務成交為 [X]件)
- [年份]年1-[月份]月回收[X]台，相較[去年]年1-[月份]月回收[X]台 (成長[X]%)

## 策略執行
- [執行項目1] - [進度說明]
- [執行項目2] - [進度說明]
- [執行項目3] - [進度說明]
```

**企業業務**

```markdown
## 衡量指標
- 成交率與業務量提升

## 策略執行
- [客戶名稱][案件說明]（PO已取得總額約[X]萬元），預計[時間]執行完畢
- [設備/服務]PO順利取得，[時間]前全數交貨，每套毛利約[X]萬
- [其他執行項目]
```

**新業務**

